WebXR viewer-pose recorder

Enters an immersive-vr session and shows a head-locked panel printing getViewerPose() in both local and local-floor (position + orientation), updated every frame. Vanilla WebXR, no libraries. On session end you can download the recorded trace and open it in the visualizer.

Downloaded as a JSON file — never rendered into the page, so large captures stay responsive. Load it in the visualizer.
